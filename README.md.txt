# QR Phishing Detection

A machine learning-based system that detects phishing URLs hidden inside QR codes.

## Features

- QR Code Generation
- URL Extraction
- Feature Engineering
- Machine Learning Classification
- Safe/Phishing Detection

## Technologies

- Python
- Scikit-Learn
- Pandas
- NumPy
- OpenCV

## Project Structure

```text
QR-Phishing-Detection/
├── src/
├── dataset/
├── models/
├── images/
├── app.py
```

## Run

```bash
pip install -r requirements.txt
python app.py
```
